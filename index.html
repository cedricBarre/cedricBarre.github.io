Franky

Franky is a robot I am currently developing that uses computer vision to make a 2D map of a room. It runs on a Raspberry Pi 3 and makes extensive use of the OpenCV Python library.

Visual Music Representation

Visual Music Representation is an interactive and flexible software that allows you to analyse repetition in music pieces with the help of Processing. This was my first time using Java to build an application and it was done for an academic project with 2 of my friends.
